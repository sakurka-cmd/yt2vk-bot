# yt2vk-bot

Бот для сохранения видео с YouTube в альбомы сообщества ВКонтакте.

## Возможности

- Подписка на YouTube-каналы с автоматической загрузкой новых видео
- Разовая загрузка видео по ссылке
- Выбор качества: 480p, 720p (по умолчанию), 1080p, 4K
- Автоматическое создание альбомов для каждого канала
- Дедупликация — одно видео не загружается дважды

## Быстрый старт (Docker)

```bash
git clone <repo>
cd yt2vk-bot

cp .env.example .env
nano .env  # Заполнить VK_TOKEN и VK_GROUP_ID

docker-compose up -d
docker-compose logs -f
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Начало работы |
| `/subscribe` | Подписка на YouTube-канал |
| `/dl <url>` | Разовая загрузка видео |
| `/list` | Список подписок |
| `/unsub` | Отписаться от канала |
| `/quality` | Изменить качество подписки |
| `/albums` | Список альбомов |
| `/addalbum` | Создать альбом |
| `/status` | Статус загрузки |
| `/help` | Помощь |

## Получение токена VK

1. Перейдите в [управление сообществом](https://vk.com/group.php?act=settings)
2. Раздел **Работа с API** → **Токен группы**
3. Выберите права: **Видео** (видео), **Сообщения** (сообщения), **Стены** (стены)
4. Сгенерируйте токен
5. Вставьте в `.env` как `VK_TOKEN`

## Требования к сообществу

- Сообщество должно быть **группой** (не публичная страница)
- У бота должны быть права администратора
- Должны быть включены **сообщения** и **видео** в настройках

## Как работает

```
YouTube → yt-dlp (скачать) → tmpfs (RAM) → VK Video API → Альбом сообщества
```

Оптимизация места:
- Видео скачивается в tmpfs (в RAM), не на диск
- Сразу после загрузки в VK файл удаляется
- Выбор качества позволяет контролировать размер
