# yt2vk-bot

Бот для сохранения видео с YouTube в альбомы сообщества ВКонтакте.

## Возможности

- Подписка на YouTube-каналы с автоматической загрузкой новых видео по RSS
- Разовая загрузка видео по ссылке
- Выбор качества: 480p, 720p (по умолчанию), 1080p, 4K
- Управление альбомами сообщества (создание, просмотр)
- Дедупликация — одно видео не загружается дважды
- Постоянная клавиатура с кнопками для удобного управления
- FSM-машина состояний с персистентностью в SQLite
- Отслеживание статуса загрузки

## Быстрый старт (Docker)

```bash
git clone https://github.com/sakurka-cmd/yt2vk-bot.git
cd yt2vk-bot

cp .env.example .env
nano .env  # Заполнить VK_TOKEN, VK_GROUP_ID, ADMIN_IDS

docker compose up -d
docker compose logs -f
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Начало работы (приветствие + клавиатура) |
| `/subscribe` | Подписка на YouTube-канал (FSM: ссылка → качество → альбом → подтверждение) |
| `/dl` | Разовая загрузка видео |
| `/list` | Список подписок с inline-кнопками |
| `/unsub` | Отписаться от канала |
| `/quality` | Изменить качество подписки |
| `/albums` | Список альбомов сообщества |
| `/addalbum` | Создать альбом |
| `/status` | Статус текущей загрузки |
| `/help` | Помощь |

## Переменные окружения (.env)

| Переменная | Обязательна | Описание |
|------------|-------------|----------|
| `VK_TOKEN` | Да | Токен группы VK с правами video, messages, photos |
| `VK_GROUP_ID` | Да | ID группы (без минуса) |
| `ADMIN_IDS` | Нет | Список VK ID администраторов через запятую. Если пусто — доступ для всех |
| `LOG_LEVEL` | Нет | Уровень логирования (default: `DEBUG`) |
| `CHECK_INTERVAL` | Нет | Интервал проверки подписок в секундах (default: 3600) |

## Получение токена VK

1. Перейдите в [управление сообществом](https://vk.com/group.php?act=settings)
2. Раздел **Работа с API** → **Токен группы**
3. Выберите права: **Видео**, **Сообщения**, **Фотографии** (для обложек)
4. Сгенерируйте токен
5. Вставьте в `.env` как `VK_TOKEN`

## Требования к сообществу

- Сообщество должно быть **группой** (не публичная страница)
- У бота должны быть права администратора
- Должны быть включены **сообщения** и **видео** в настройках
- Должна быть включена кнопка **«Начать»** в настройках сообщений (для нового формата VK Long Poll)

## Архитектура

```
YouTube → yt-dlp (скачать) → tmpfs (RAM) → VK Video API → Альбом сообщества
```

```
bot/
├── main.py              # Точка входа, LP-патч, запуск
├── handlers.py          # Обработчики команд и FSM-состояний
├── keyboards.py         # VK клавиатуры (постоянная + inline)
├── states.py            # Enum состояний FSM
├── state_dispenser.py   # Кастомный StateDispenser (SQLite-backed)
├── database.py          # SQLite: подписки, задачи, FSM-состояния
├── downloader.py        # yt-dlp обёртка: скачивание, извлечение ID
├── uploader.py          # VK Video API: загрузка, альбомы
├── scheduler.py          # Фоновый цикл проверки подписок
└── config.py            # Конфигурация из .env
```

## Оптимизация

- Видео скачивается в tmpfs (в RAM), не на диск
- Сразу после загрузки в VK файл удаляется
- Выбор качества позволяет контролировать размер
- FSM-состояния хранятся в SQLite — переживают перезапуск бота
